# Cryptex

An NFT platform where users authenticate via MetaMask and store/manage seed
phrases as encoded NFTs. Built the backend interfacing with the Polygon
blockchain for secure, efficient transactions.

## Stack

MetaMask, Polygon, Solidity, full-stack
