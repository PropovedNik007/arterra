# Red Cross lip-reading app

A lip-reading web app built (via Coding Austria) for the Austrian Red Cross
to aid communication in noisy environments — streams smartphone video into
the AV-HuBERT model to read lips and generate text output.

## Stack

AV-HuBERT, FastAPI, Next.js
