# Tourism knowledge graph

Tourism Technology Hackathon 2023 project: a data pipeline mapping data
from multiple sources into a knowledge graph on tourism subjects,
visualized as an interactive map with an accompanying chatbot for analyzing
points of interest. Placed 2nd Place Winner.

## Stack

Selenium, GeoPandas, Folium
